cRFC

No need any network connection, with all RFC documents after downloaded. Documents included from RFC1 to the latest RFC8401, as there is much knowledge value, for everyone IT professionals. Available in AppStore now.

Features

Smooth operation, easy to understand, and focused on the reading experience.

Support full-text searching keyword in current RFC.

Support searching keyword in all RFC documents index.

Navigate in multiple RFC opened history like browser.

Extreme compression unused documents to reduce space.

Purchase once, read forever. There is no any addition content or function which needs extra cost.

Language Explanation

All documents in cRFC are original English documents. App language support and localization support are only in the UI layer, which don’t contain any document translation.

RFC Brief Addition

In information and communications technology, a Request for Comments (RFC) is a type of publication from the technology community. RFCs may come from many bodies including from the Internet Engineering Task Force(IETF), the Internet Research Task Force (IRTF), the Internet Architecture Board (IAB) or from independent authors. RFCs have since become official documents of Internet specifications, communications protocols, procedures, and events.
